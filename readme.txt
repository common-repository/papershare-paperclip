Papershare Paperclip Wordpress Plugin

== Description ==
PaperClips let you instantly put any content you publish with PaperShare onto any website or blog. You create the PaperClip on PaperShare, and then copy an embed code onto any site where you want the content to appear.

This plugin was developed to simplify the process of adding a PaperShare PaperClip to a WordPress page, even if your WordPress setting have not enabled the use of JavaScript. The PaperClip plugin makes it easy to create reusable shortcodes that will display your PaperClip on a WordPress page page.


== Installation ==
**From your Wordpress dashboard**

1. Visit 'Plugins > Add New'
2. Search for 'PaperShare'
3. Activate PaperShare from your Plugins page.
4. Visit 'Settings > PaperClips' and create some PaperClip Shortcodes

**From Wordpress.org**

1. Download PaperShare
2. Unzip the PaperShare zip file
3. Upload the 'PaperShare' zip file to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
4. Activate PaperShare from your Plugins page.
5. Visit 'Settings > PaperClips' and create some Paperclip Shortcodes

Alternate Download Option
Download it [here](http://papershare.com/WordPressPaperClipsPlugin "PaperShare"). Then follow the directions outlined above.
